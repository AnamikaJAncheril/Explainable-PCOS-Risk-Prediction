# Explainable AI-Based PCOS Risk Assessment System

## Overview
This project predicts PCOS risk using Machine Learning and Explainable AI.

## Features
- Random Forest Classifier
- 92.4% Accuracy
- SHAP Explainability
- Feature Importance Analysis
- Personalized Recommendations

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- SHAP
- Matplotlib
- Seaborn

## Results
- Accuracy: 92.4%
- Precision: 95%
- F1 Score: 86%

## Key Predictors
- Follicle No. (R)
- Follicle No. (L)
- Weight Gain
- Hair Growth
- Skin Darkening
- Cycle Length
